# Tierdom Security Reports

Note that no fully automated AI submissions are acceptable.
AI-assisted is fine, but there must always be a human in the loop, who takes ownership and responsibility for the report.

For regular security concerns that can be handled through community discussion and bugfixing, open an issue on [the Tierdom-App GitHub repository](https://github.com/tierdom/tierdom-app).

For major issues that require responsible, private disclosure, contact the core maintainer directly. Find ways to do so at [jeroenheijmans.nl](https://www.jeroenheijmans.nl/).
